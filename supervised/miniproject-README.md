# 🎮 Python Rock Paper Scissors Game

A simple command-line Rock–Paper–Scissors game built with Python where a user plays against the computer.

This project is beginner-friendly and demonstrates Python basics such as user input, conditional statements, lists, and the random module.

## 📌 Features

- Play Rock–Paper–Scissors against the computer
- Computer move is generated randomly
- Displays the computer's choice
- Announces whether the player wins, loses, or draws
- Simple command-line interface

## 🛠 Technologies Used

- Python 3
- Random Module

## ▶️ How to Run

```bash
python game.py
