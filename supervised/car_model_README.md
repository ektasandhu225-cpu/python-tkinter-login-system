# Car Model Prediction (Supervised Learning)

## Project Description
This project demonstrates a **Supervised Machine Learning model** used to analyze and predict car related data using Python.

The notebook performs data preprocessing, model training, and prediction using machine learning algorithms.

## Files Included
- car model.ipynb : Main notebook containing the machine learning model
- Register(page).ipynb : Registration related notebook

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Machine Learning Type
Supervised Learning

## Steps Performed
1. Import required libraries
2. Load dataset
3. Data preprocessing
4. Train machine learning model
5. Evaluate model performance
6. Predict results

## How to Run
1. Open the notebook in **Jupyter Notebook**
2. Run all the cells step by step
3. The model will train and show the output
